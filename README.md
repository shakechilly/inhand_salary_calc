# CTC → In-Hand Salary Calculator Interactive salary calculator for India's New Tax Regime FY 2025-26. ## Features - Annual CTC to Monthly In-Hand estimation - New Regime tax slabs - Employee PF support - Employer PF support - Fixed PF cap mode (₹1,800/month) - NPS deductions - Tax breakdown table - Interactive graph - Piecewise function display --- ## PF Fixed Mode When enabled: Employee PF = ₹21,600/year Employer PF = ₹21,600/year Total PF impact = ₹43,200/year Taxable Income: Taxable Income = CTC − ₹21,600 − NPS − ₹75,000 Monthly Cash Flow: PreTax Monthly = ( CTC − ₹21,600 − ₹21,600 − NPS ) / 12 --- ## Installation npm install npm run dev --- ## Build npm run build --- ## Tech Stack - React - Vite - HTML Canvas
