# CTC → In-Hand Salary Calculator

Interactive salary calculator for India's New Tax Regime FY 2025-26.

## Features

- Annual CTC to Monthly In-Hand
- Employee PF
- Employer PF
- NPS
- Interactive graph
- Tax slab breakdown
- Piecewise function view

## PF Fixed Mode

When enabled:

Employee PF = ₹21,600/year

Employer PF = ₹21,600/year

Total PF impact = ₹43,200/year

Taxable Income:

CTC
− ₹21,600
− NPS
− ₹75,000
